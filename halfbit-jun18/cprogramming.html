C++ was developed by Bjarne Stroustrup. In 1985, the first edition of the C++ Programming Edition was announced to the world. It was continuously updated to fix and new features and changes.

C++ is an object-oriented language. It has a number of new features as compared to C such as Abstraction, Polymorphism, Inheritance and many others. C++ soon went popular amongst programmers and as of 2017, it is the third-most popular programming language all over the world.

C++ also has data structures which means organizating, managing and storing all the data in different ways such as Trees, Heaps, Linked-Lists, Arrays, Stacks, Queues etc.

In this course, we'll cover all the basics of C and C++ which will make your fundamentals strong and help you in other programming languages as well.
